## Conceptos Básicos

JL Planner es un sistema de gestión de proyectos basado en diagramas de Gantt que permite planificar, organizar y hacer seguimiento de proyectos complejos mediante una estructura jerárquica de elementos.

El sistema utiliza un enfoque orientado a objetos donde cada elemento del proyecto (tareas, hitos, procesos) hereda de una clase base común pero tiene comportamientos específicos según su tipo.
